This example shows:
  • Validation of fields
    • Mandatory fields can't remain empty
    • Some fields like dates have custom rules
    • validation rules can depend on values of other fields
    • Option to surpress validation in some fields (try past date in Value Field)
  • Form with many dependencies between fields
    • Depending on the chosen CCY, the NDF checkbox becomes enabled or enabled and checked
    • Select CCY like ARS - and the NDF checkbox becomes enabled
  • Dynamic fields
    • If NDF is checked, 2 fields will be display: Fixing date and index
    • If SWAP is chosen, 4 more fields will be shown
    • Change the top radio button to FX Option
    • Change contract type from Vanilla and see how different fields are shown
  • Number formatting with AutoNumeric.js
  • Clear form button after submission
